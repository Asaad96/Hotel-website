## 📌 About
A responsive hotel landing page built as part of my self-learning journey in frontend development.
The project focuses on clean layout, responsive design, and semantic HTML structure.

## ✨ Features
- Fully responsive layout (mobile + desktop)
- Clean navigation and hero section
- Rooms & services showcase
- Contact / booking section

- CSS3
